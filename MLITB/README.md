(Convolutional) Neural Network
==============================

####Implemented Layer :

- Convolution Layer

- Pooling Layer

- Full-Connected Layer

- Activation of Hidden Layer

  - Sigmoid

  - Relu

  - Linear

- Activation of Output Layer

  - Sigmoid

  - Linear/Regression

  - Softmax

####Implemented Trainer :

- Gradient Descent

  Parameter :

  - batch_size (1 for Stochastic Gradient Descent)

  - learning_rate

  - momentum
